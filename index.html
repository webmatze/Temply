About Temply.js

Temply.js is a small templating engine using vanilla JavaScript.

It does not depend on any other framework such as jQuery, Prototype.js or other. But it can be used with any of them.

Requirements

Temply.js should work with most modern browsers. It has been tested with this following browsers:

  • Firefox 18 (Mac)
  • Safari 6 (Mac)
  • Google Chrome 23 (Mac)

But other versions or browsers could work, too.

Usage

Temply.js uses a simple template language that can be used in two different ways.

Text Template Tags

This is the simplest way to include Temply.js template tags into your HTML pages.

You can include the template tags as {{name}} nearly everywhere in your HTML page.

<div id="data">
  <h1>{{title}}</h1>
  <p>{{content}}</p>
  <a href="{{url}}">{{link_name}}</a>
</div>

Tag Attribute Template Tags

A second and cleaner way is to use the following attribute tp-model="name" on any HTML tag. But this can not be used to set the values of other tag attributes like href="" in the following example. But you can use the Text Template Tag here.

<div id="data">
  <h1 tp-model="title"></h1>
  <p tp-model="content"></p>
  <a href="{{url}}" tp-model="link_name"></a>
</div>

Replacing Template Tags With Data

When all template tags are placed you can replace those tags using Temply.js with dynamic data:

var data = {
  title: "Hello World",
  content: "Lorem ipsum...",
  url: "http://example.com",
  link_name: "Example.com"
};
var temply = new Temply(data, document.getElementById("data"));
temply.run();
